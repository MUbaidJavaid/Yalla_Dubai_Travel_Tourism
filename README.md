# Travel Booking System

## 🌍 Project Overview
This is a web-based travel booking platform that allows users to browse tours, make reservations, and process payments. Admins can manage tours, view orders, and update product details through an easy-to-use dashboard.

## ✨ Key Features
- **User accounts** - Sign up, login, and manage bookings  
- **Tour browsing** - View available tours by city or category  
- **Booking system** - Reserve private or shared tours with payment processing  
- **Admin dashboard** - Manage tours, view orders, and update inventory  
- **Shopping cart** - Save tours before checkout  
- **Discount system** - Automatic discounts applied at checkout  

## 🛠️ Setup Guide

### Prerequisites
- Node.js (v14 or higher)  
- MongoDB (running locally or cloud connection)  
- Cloudinary account (for image storage)  

### Installation
1. Clone the project  
2. Run `npm install` to install dependencies  
3. Create a `.env` file with your configuration (ask developer for template)  

### Running the Application
```bash
npm start
